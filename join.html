Створимо цей сайт разом!

Цей сайт створено для спільного наповнення, тому Кожен бажаючий може долучитися. Наша мета створити місце, де буде згрупована вся найважливіша і лише перевірена інформація.

1. Додати інформацію листом

Цей спосіб найлегший, але разом з цим і найдовщий, з точки зору потрапляння на сайт

  1. Надсилайте нам інформацію на 4uainfo@gmail.com

2. Додати інформацію через Соцмережі

Учасники Соцмереж відкрито обговорюють запропоновані зміни і чекають оновлення

  1. Приєднуйтесь до нас в facebook
  2. Надсилайте інформацію в приватних повідомленнях або публікуючи на стіні

3. Додати інформацію через GitHub

Цей спосіб передбачає наявність специфічних знань, але є найбільш прозорим і швидшим

  1. Зареєструйтесь на Github
  2. Перейдіть на https://github.com/4ua/4ua
  3. Внесіть зміни шляхом створення Pull Request
    • Весь контент сайту знаходиться в файлах index.html (Постраждали), missing.html (Зникли), join.html(Долучитися) та faq.html(?).
    • Відкрийте той файл, в який потрібно внести зміни та натисніть Edit.
    • Після внесення змін коротко опишіть їх суть в Commit summary (наприклад: "Added new info about Kosak Mamay")
    • Додайте більше коментарів, якщо такі знайдуться і натисніть Propose File Change
  4. З цього моменту всі бажаючі можуть стежити за процесом розгляду і затвердження запропонованої зміни
